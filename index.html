Blåhaj is a discord bot that was inspired by one of Ikea's product, the blahaj soft toy.

Blåhaj is a utility bot that can send images of blåhajs and much more!
Some Features

Sends Images

When commanded, blahaj will send images of blahaj from the internet to the server.

Server Welcome

when a new user joins the server, blahaj will welcome the user.

Use the command and Blåhaj will flip a coin and give you the result
